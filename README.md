# Python Day 1 Demo - Superhero Name Generator

This is a simple browser-based demo for Day 1 of a beginner Python course.

Students can try the finished project before building it themselves.

## Live Demo
Enable GitHub Pages in the repository settings to get a live link.

## Project Idea
The program asks:
1. What city did you grow up in?
2. What is your favorite animal?

Then it combines them to generate a superhero name.

Example:
Tokyo + Falcon = Tokyo Falcon


Try it yourself here: https://sentakkuofficial.github.io/python-lesson1-superhero-demo/
